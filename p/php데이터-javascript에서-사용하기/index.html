php데이터 javascript에서 사용하기

Jan 13, 2022
1 minute read

요약 :
php 변수에 데이터를 담고 script 에서 javascript 변수로 옮김

받을 데이터가 문자나 숫자 같은 원시데이터 일때

// php 변수에 해당 데이터를 담는다.
<!--@$youtubeId = $main->VAR['banner']['main-youtube']['list']['ko']['list'][0]['url']; -->

// 스크립트 태그 내부에 php 변수를 <?=변수명?> 으로 담아서 사용
const youtubeId = "<?=$youtubeId?>";

받을 데이터가 객체일때

// 위와 마찬가지로 php 변수에 해당 객체를 담은후에 json으로 받는다.
const recomCn = <?=json_encode($recomCn)?>;

// 리스트에 적용할시 객체 형식이기에 length 를 확인하기 위해선 Object.keys()를 사용한다.
Object.keys($recomCn[0]).length  // 객체의 키 갯수 확인하기

Memo

  • 스크립트에 데이터가 노출되니 중요하지 않은 데이터 이외에는 사용하면 안됨
